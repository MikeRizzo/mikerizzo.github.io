A skeleton to copy when starting a post. The front matter above is the minimum
worth setting: `category: blog` puts it on the Blog page, and `author` fills the
block at the bottom. Set `projects: true` instead to route it to Projects, or
`star: true` to mark it in the listing.

## Headings and copy

Second and third level headings carry the structure of a post.

### A third level heading

Body copy, with **bold**, *italic*, and a [link](https://github.com/MikeRizzo).
Inline `code` sits on a subtle chip.

## Code

```python
import torch

def accuracy(logits, labels):
    preds = logits.argmax(dim=-1)
    return (preds == labels).float().mean().item()
```
