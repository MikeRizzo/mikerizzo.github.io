## Code

Fenced blocks are highlighted with Rouge, using the Monokai palette.

```python
class Trainer:
    def __init__(self, model, lr=1e-3):
        self.model = model
        self.lr = lr

    def step(self, batch):
        loss = self.model.loss(batch)
        print(f"loss={loss:.4f}")
        return loss
```
